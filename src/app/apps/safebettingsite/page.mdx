Site built to be deployed in the wordpress platform. Something that was taken into considaration was that the site had to be easy to use and maintain.

This initial site is geared towards small betting companies or agents that want to have a web presence.

The Front-End is built with WordPress, the database used is MySQL, for the static images a Digital OCean Spaces is used, and for the map a Mapbox API and Leaflet is used.


The most importat components in this template are the Map, since it can be used in pretty much any other template, and the Dashboard, since it allows the user to manage the content of the website.
